# Sudoku-Project
   Sudoku Project For AP COMPUTER SCIENCE
   ATEF ISTIAQUE-
   My Sudoku Project, it uses the idea of messing up a default solution to generate new ones.-
   How to run the program: Load the code into VSCode or JavaSandbox or something and press run.-
   I take a default solution, and swap rows or columns randomly, the program checks to see if a swap is valid and if it's not, then the swap is undone and a new one is generated.-
